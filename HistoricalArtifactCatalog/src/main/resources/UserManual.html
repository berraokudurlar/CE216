üí° Tips

  • Select an artifact with a single-click; deselect with a double-click.
  • Use filters on the left panel and click Apply Filter.
  • Use the File menu to Import/Export artifacts as JSON files.
  • Make sure to save your work before exiting or importing.
üíæ Saving Your Work: HistoFact does not auto-save. To preserve your data, regularly use the Export JSON option from the File menu. This saves your current artifacts as a backup. You can later restore them by selecting Import JSON. Always export before exiting or making major changes.
